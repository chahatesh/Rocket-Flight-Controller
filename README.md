# 🚀 Rocket Flight Controller + Live 3D Telemetry Dashboard

A real-time rocket flight telemetry system built around ESP32 microcontrollers and a custom web-based ground station featuring a live 3D orientation model, altitude tracking, and flight analysis tools.

This project brings together embedded systems, wireless communication, and real-time visualization to replicate core concepts used in modern aerospace telemetry systems.

---

## 🌌 Overview

During flight, the onboard ESP32 continuously collects motion and altitude data from sensors and streams it to a ground station. The data is visualized in real time through a custom web dashboard that includes:

- A live 3D model that rotates based on real sensor orientation
- Real-time altitude tracking
- Apogee (maximum height) detection
- Flight motion visualization

This creates a direct link between physical movement and digital representation, allowing full visibility into rocket behavior during flight.
